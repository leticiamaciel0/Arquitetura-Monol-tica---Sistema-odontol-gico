# Arquitetura-Monol-tica---Sistema-odontol-gico
O sistema é um programa simples em Python que organiza a rotina de uma clínica odontológica. Ele permite cadastrar pacientes, registrar prontuários, agendar consultas, controlar dentistas e registrar pagamentos. Tudo funciona em um único sistema (arquitetura monolítica) e é acessado por um menu no terminal.
